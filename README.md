# scrapy_spider
学习网络爬虫  

## 基本内容
本代码库记录了学习编写简单网络爬虫的相关过程，供以后参考：  
+ 学习了requests库的使用方法
+ 学习了Beautiful库的使用方法
+ 利用上述两个代码库，分别爬取了链家网站上*苏州相城区*、*北京西城区*二手房房价数据，并进行简单处理可视化
